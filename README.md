# Stock Manager Project

## Algorithms Used

### Merge Sort
Purpose:
Sorts inventory items alphabetically.

Time Complexity:
O(n log n)

Reason:
The list is repeatedly divided into smaller halves and merged back together in sorted order.

---

### Binary Search
Purpose:
Efficiently searches for inventory items by name.

Time Complexity:
O(log n)

Reason:
The search range is cut in half during each step of the algorithm.

---
